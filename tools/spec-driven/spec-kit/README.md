**Stars:** 83K+ | **by GitHub**

## 简介

面向规范驱动开发（Spec-Driven Development）的开源工具集。核心理念是把 spec（规范）变成可执行的，直接驱动实现，而非仅仅作为参考文档。

- **speckit.analyze** — 分析需求，梳理边界
- **speckit.clarify** — 澄清模糊点，确认约束
- **speckit.design** — 生成数据结构、API 接口、系统设计
- **Specify CLI** — 命令行工具，驱动完整 spec → implement 流程
- **Extensions & Presets** — 社区扩展与预设，支持定制化工作流

支持 GitHub Copilot、Claude Code 等主流 AI agent。

## 工作流

```
speckit.analyze → speckit.clarify → speckit.design → implement
```

先定义清楚「做什么」，再交给执行工具（如 Superpowers）完成「怎么做」。
