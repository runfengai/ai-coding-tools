# everything-claude-code

**Repo:** https://github.com/affaan-m/everything-claude-code

**Stars:** 50K+ | **Anthropic Hackathon Winner**

## 简介

面向 AI agent harness（Claude Code、Codex、Opencode、Cursor 等）的性能优化系统。不只是配置文件，而是一套完整体系：

- **Skills & Instincts** — 可复用的技能与直觉规则
- **Memory Optimization** — 跨 session 自动保存/加载上下文的 hooks
- **Continuous Learning** — 从会话中自动提取模式，沉淀为可复用技能
- **Security Scanning** — AgentShield 安全扫描，防护 prompt injection 等攻击
- **MCP Configurations** — 生产级 MCP 配置
- **Verification Loops** — checkpoint & continuous evals，grader 类型，pass@k 指标

经过 10+ 个月密集日常使用、构建真实产品后演化而来。

## 适用场景

- 优化 Claude Code agent 的 token 使用与响应质量
- 跨 session 持久化 memory
- 为 AI agent 添加安全防护
- 并行化 agent 工作流（git worktrees、cascade method）
